# tidesurf projects
Demonstration of [tidesurf](https://github.com/janschleicher/tidesurf) on publicly available scRNA-seq datasets.

Parquet files containing the results of the various performed analyses are available in this GitHub repository in the [output folder](out).
Raw count matrices generated with tidesurf or the other tools against which we compare it as well as processed AnnData objects are available from Zenodo under the following DOI: [10.5281/zenodo.19096001](https://doi.org/10.5281/zenodo.19096001).
